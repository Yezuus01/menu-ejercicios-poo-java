# Sistema de Menú de Ejercicios con POO en Java

## Descripción
Aplicación interactiva que gestiona 16 ejercicios de lógica y programación, aplicando conceptos fundamentales de **Programación Orientada a Objetos (POO)** como herencia, abstracción y polimorfismo. El sistema utiliza `JOptionPane` para una interfaz visual sencilla.
Proyecto elaborado por **Jesus Condori Aruquipa** -- **INFORMATICA INDUSTRIAL - TERCER SEMESTRE - 2026**
## Cómo ejecutar
1. Clona el repositorio.
2. Abre el proyecto en tu IDE preferido (IntelliJ IDEA, VS Code, etc.).
3. Ejecuta la clase `Main.java`.

## Ejercicios Incluidos
1. **Invertir una Cadena**: Invierte el texto ingresado.
2. **Verificar Palíndromo**: Revisa si una palabra se lee igual en ambos sentidos.
3. **Contar Vocales y Consonantes**: Análisis de caracteres en frases.
4. **Invertir un Entero**: Lógica matemática para voltear números.
5. **Mayor de Tres Números**: Comparaciones lógicas.
6. **Intercambiar Números**: Cambia los valores de dos variables sin usar una tercera.
7. **Encontrar Duplicados**: Identifica los elementos repetidos dentro de un arreglo.
8. **Verificar Anagramas**: Comprueba si dos palabras contienen las mismas letras.
9. **Segundo Número Más Grande**: Encuentra el segundo valor más alto en una lista.
10. **Serie de Fibonacci**: Genera la secuencia matemática hasta el término indicado.
11. **Verificar Número Primo**: Determina si un número es divisible solo por sí mismo y por uno.
12. **Factorial de un Número**: Calcula el producto acumulado de todos los números hasta N.
13. **Contar Ocurrencias**: Cuenta cuántas veces aparece un carácter específico en un texto.
14. **Eliminar Duplicados**: Crea una lista nueva quitando los elementos repetidos.
15. **Suma de Dígitos**: Suma cada una de las cifras que componen un número entero.
16. **Número Armstrong**: Validación de potencias de dígitos.

## Tecnologías Usadas
* Java JDK 11+
* Swing (JOptionPane) para interfaz gráfica.
* Git/GitHub para control de versiones.
